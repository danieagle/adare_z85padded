
Adare_Z85padded
=====================

Version: 0.0.1-dev

ZeroMQ z85 encoding with allways padding.

Before Version 1.0.0 this is a work in progress.


Maintenance Status: Very Active.

Adare_Z85padded is written in Ada2012 and C.

It need Alire to build.

Adare_Z85padded crate will be published in Public Alire after 1.2.0 version. Enjoy!!

How a Bonus Almost all packages in this lib is "Ada Pure"! :-)

USAGE
-----

  Building before publishing in Public Alire Arquive:

    1) enter in your alire project:
    2) alr with adare_z85padded --use https://github.com/danieagle/adare_z85padded.git

    3.1) mybe use: alr build --release
      or
    3.2) mybe use: alr build --development
      or
    3.3) mybe use: alr build --validation

  And Be Happy :-D

  Main repository => https://github.com/danieagle/adare_z85padded.git

  See file AlreadyTestedIn.txt

  Enjoy!!

For documentation, see Doc/adare_z85padded.pdf

For examples and how compile in your own projects, see:
  src/*.ad{s,b}
  adare_z85padded_examples_01/src/*.adb

  Enjoy!!
